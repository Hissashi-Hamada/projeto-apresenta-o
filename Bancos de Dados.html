Banco de Dados

Banco de dados são sistemas que armazenam, organizam e gerenciam dados de forma eficiente. Eles são fundamentais para aplicações web, permitindo o armazenamento de informações como usuários, produtos, transações e muito mais. Existem diferentes tipos de bancos de dados, como relacionais (SQL) e não relacionais (NoSQL), cada um com suas características e usos específicos.

Oracle Database

Tipo: Relacional (SQL)
Uso: Corporativo, bancos, grandes empresas
Motivo: Super robusto, seguro, com alta escalabilidade e suporte empresarial.
Usado por: grandes empresas financeiras, governo, telecom.


Linguagens usadas na construção:
C e C++ – base do núcleo (engine do banco)
Java – usado nas ferramentas gráficas e gerenciadores
PL/SQL – linguagem procedural própria da Oracle (tipo um SQL turbinado)
Linguagens usadas para interagir com o banco:
Java (via JDBC)
Python (via cx_Oracle)
C# / .NET
PHP, Node.js, Go, Ruby, etc.
A Oracle tem suporte a praticamente todas as linguagens populares — ela é super focada em grandes empresas que usam linguagens variadas.

MySQL

Tipo: Relacional (SQL)
Uso: Web, WordPress, e-commerce
Motivo: Gratuito, confiável e leve. Suporta desde projetos pequenos até sistemas grandes.
Usado por: o WordPress, Joomla, e boa parte da internet.


Linguagens usadas na construção:
C e C++ – base de todo o sistema do MySQL
Assembler (em partes antigas do core)
Algumas interfaces e ferramentas usam Python ou JavaScript
Linguagens usadas para interagir com o banco:
PHP (muito usado com LAMP – Linux, Apache, MySQL, PHP)
JavaScript (Node.js) (mysql2, sequelize)
Python (mysql-connector, SQLAlchemy)
Java (JDBC, Hibernate)
C# / .NET
Ruby, Go, Perl, e muitas outras
